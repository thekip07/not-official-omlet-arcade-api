OmletAPI - alternative OmAPI

OmletAPI allows you to interact with the Omlet Arcade platform,
get various information about the streamer. OmletAPI is not official
API by Omlet Arcade. You don't need to get a token for private
methods, just specify your password in the &password parameter.
